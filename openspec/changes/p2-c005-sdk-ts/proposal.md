# p2-c005 — TypeScript SDK

## Phase
phase-2-generated-sdks

## Depends on
p2-c003 (generated TS stubs must exist under `sdks/ts/src/gen/`)

## Directory
`sdks/ts/`

## What this change does

Creates the TypeScript SDK package on top of the generated Connect-ES stubs.

### Package structure

```
sdks/ts/
├── package.json              # @prometheusags/frf-sdk
├── tsconfig.json
├── src/
│   ├── index.ts              # re-exports SpineClient, generated types
│   ├── client.ts             # SpineClient class
│   └── gen/                  # generated by buf (created in p2-c003)
│       └── flint/v1/         # *.ts stubs
└── dist/                     # compiled output (ESM + CJS, not committed)
```

### `src/client.ts`
