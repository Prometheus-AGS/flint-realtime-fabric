# p2-c004 — Go SDK

## Phase
phase-2-generated-sdks

## Depends on
p2-c003 (generated .pb.go + _connect.pb.go files must exist)

## Directory
`sdks/go/`

## What this change does

Creates the Go SDK module on top of the generated protobuf + Connect stubs.

### Module structure

```
sdks/go/
├── go.mod                    # module github.com/prometheusags/frf/sdks/go
├── go.sum
├── gen/                      # generated by buf (created in p2-c003)
│   └── flint/v1/             # *.pb.go + *connect.pb.go
├── client/
│   ├── spine_client.go       # SpineClient wrapper
│   └── smoke_test.go         # //go:build integration
└── README.md
```

### `client/spine_client.go`

```go
package client

type SpineClient struct {
    conn connectrpc.Client
}

func New(baseURL string, opts ...connect.ClientOption) *SpineClient
func (c *SpineClient) Publish(ctx context.Context, envelope *flintv1.EventEnvelope) (*flintv1.Offset, error)
func (c *SpineClient) Subscribe(ctx context.Context, req *flintv1.SubscribeRequest) (iter.Seq[*flintv1.EventEnvelope], error)
```

### Dependencies

```
connectrpc.com/connect@v1.20.0
google.golang.org/protobuf@v1.36.11
```

## Exit criteria

- `go build ./...` from `sdks/go/` exits 0
- `go vet ./...` exits 0
- `go test ./...` (non-integration) exits 0 (no unit tests required beyond compilation)
- Integration test file compiles with `//go:build integration` tag
